# Dots and Boxes – Documentation Catalogue for Full Lifecycle

## 1. 비전·비즈니스·기획 단계 문서
- **제품 비전 문서 (Product Vision & Strategy)**
- **타깃 유저 정의 및 페르소나 문서**
- **시장·경쟁 분석 리포트**
- **비즈니스 모델·수익 구조 캔버스**
- **성공 지표·KPI 정의 문서**
- **하이레벨 로드맵(연·분기 단위)**

## 2. 콘셉트·게임 디자인 문서
- **게임 콘셉트 원페이저(One‑pager)**
- **게임 디자인 문서(GDD, Game Design Document)**  
  - 규칙·승리 조건·모드(싱글, 멀티, 튜토리얼)  
  - 체인/루프, 난이도, 변형 규칙(Dots and Boxes 고급 전략 포함)
- **레벨/난이도 설계 문서** (초등·유아/고급자 분리)
- **경제 시스템·메타게임 설계**(코스메틱, 랭킹 등 선택)
- **UX 원칙·톤 앤 매너 가이드**

## 3. 요구사항·제품 명세
- **비즈니스 요구사항 문서(BRD)**
- **제품 요구사항 문서(PRD)**  
  - 유즈케이스·유저 스토리·액셉턴스 크라이테리아  
  - 기능/비기능 요구사항(성능, 지연, 가용성, 보안 등)
- **소프트웨어 요구사항 명세(SRS)**
- **시스템 제약사항·규제·개인정보 요구사항 정리 문서**

## 4. UX / UI 설계 문서
- **IA(Information Architecture) & 네비게이션 맵**
- **사용자 플로우(User Flow) 다이어그램**  
  - 온보딩 → 튜토리얼 → 연습 모드 → 대전/랭킹
- **와이어프레임 세트 (모바일/웹 뷰)**
- **인터랙션 설계**(제스처, 애니메이션, 피드백 패턴)
- **UI 스타일 가이드**(타이포, 컬러, 컴포넌트 라이브러리)
- **접근성 가이드**(색약, 글자 크기, 사운드 피드백 등)

## 5. 게임 로직·AI·알고리즘 설계
- **Dots and Boxes 규칙·전략 명세**(체인/루프/이중교차 등)
- **AI 난이도 설계 문서**(초급/중급/고급 봇 전략)
- **턴·상태 머신 정의(State Machine Spec)**
- **점수 계산·매치메이킹·랭킹 알고리즘 명세**
- **랜덤성·RNG 설계 문서**(시드, 재현성, 공정성)

## 6. 아키텍처·기술 설계
- **시스템 아키텍처 문서 (High‑level Architecture)**
- **레이어 구조·프레임워크 선택 정리**  
  - 클라이언트(예: Flutter/React/Unity 등)  
  - 백엔드(예: Node.js/Nest, Go, Spring 등)  
  - 실시간 통신(WebSocket, gRPC 등)
- **모듈·서비스 분해 설계**(게임 서버, 매치메이킹, 인증, 랭킹 등)
- **기술 스택·버전·코딩 컨벤션 문서**
- **API 설계 문서(Swagger/OpenAPI 명세 포함)**
- **데이터 흐름도(DFD), 시퀀스 다이어그램(UML)**

## 7. 데이터 모델·스토리지
- **ERD (Entity Relationship Diagram)**  
  - User, Profile, Game, Move, BoardState, Match, Ranking, TutorialProgress 등
- **스키마 정의 문서(SQL/NoSQL)**
- **데이터 수명주기·아카이브·삭제 정책 문서**
- **분석 이벤트 스키마 정의**(게임 로그, 튜토리얼 이탈, 승률 등)

## 8. 워크플로·프로세스 문서
- **개발 프로세스(SDLC) 정의**: Agile/Scrum/Kanban
- **릴리즈·배포 파이프라인 정의(CI/CD 플로우)**
- **코드 리뷰·브랜치 전략 문서**(Git flow, trunk 기반 등)
- **이슈 트래킹·우선순위 정책 문서**
- **실시간 운영 플로우 문서**(장애 대응, 롤백 절차)

## 9. 인프라·운영 설계
- **인프라 구조도**(클라우드 리전, VPC, 서브넷, LB, CDN 등)
- **환경 구성 문서**(Dev/Stage/Prod)
- **컨테이너·오케스트레이션 설계**(Kubernetes 등 선택 시)
- **모니터링·로깅·알림 설계 문서** (APM, 로그 수집, 대시보드)
- **성능·부하 테스트 계획**
- **보안 아키텍처 문서**(인증·인가, 비밀관리, DDoS, 레이트 리밋)

## 10. 품질·테스트 문서
